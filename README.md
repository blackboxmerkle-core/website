# Black Box Merkle — Website

Official website for [Black Box Merkle Ltd](https://blackboxmerkle.com) — cryptographic infrastructure for verifiable trust.

## About

Black Box Merkle is a Scottish technology company focused on Merkle tree systems, zero-knowledge proofs, and verifiable data structures.

**Registered Company:** SC881296  
**Registered Address:** 5 South Charlotte Street, Edinburgh, Scotland, EH2 4AN

## Stack

- Pure HTML, CSS, JavaScript — no framework dependencies
- Google Fonts (Syne + Space Mono)
- Fully responsive
- Custom cursor, scroll-triggered animations, intersection observer reveals

## Development

Just open `index.html` in a browser — no build step required.

```bash
# Or serve locally with any static server, e.g.:
npx serve .
# or
python3 -m http.server 8080
```
